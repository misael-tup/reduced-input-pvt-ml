# Examples Directory

Public examples demonstrating the usage of the `pvt_ml` framework.

- `literature_dataset_overview.py`: A safe, executable script that summarizes and validates the public Literature Validation Set. It performs an aggregate overview without printing sensitive row-by-row data.
- **No proprietary data** will ever be used in these examples.
- Any future example scripts must be fully executable using only approved public datasets or data-free templates.
