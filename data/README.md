# Data Directory

This directory is strictly reserved for **public, traceable, and approved datasets**.

- The directory contains the **Literature Validation Set** (`literature_validation.csv` and `literature_sources.csv`).
- These datasets are public data derived exclusively from published literature sources.
- **Proprietary Mexican PVT datasets are strictly excluded** from this repository due to confidentiality agreements.
- The files here have been strictly validated with `scripts/validate_literature_dataset.py` to ensure no confidential or proprietary data points are leaked.
