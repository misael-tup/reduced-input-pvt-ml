# Models Directory

This directory is reserved for publicly releasable machine learning model artifacts.

- **Trained models (`.joblib`, `.pkl`, etc.) derived from proprietary data (including the Development Set) are strictly prohibited** and will not be published.
- Any future models placed in this directory will be demonstrative only, trained exclusively on approved public datasets, or explicitly authorized for release.
- A demonstrative model should not be interpreted or presented as the final proprietary model used in the paper.
