# Literature Validation Set

*(The dataset is incorporated in the `data/` directory, pending final review before publication of the repository.)*

The Literature Validation Set is the main public dataset for this repository. 

- It originates entirely from published, publicly available sources.
- The datasets were generated from a pointwise literature table, meticulously filtered to only retain the experimental data, fully purging any predictions or errors from proprietary models.
- It includes full traceability, indicating the specific fluid, source publication, units, columns, applied filters, and corresponding references.
- **Under no circumstances** is this dataset mixed with proprietary Mexican PVT datasets.
- The CSVs strictly adhere to the mandatory schema documented in [literature_schema.md](literature_schema.md).
