# Data Availability

Due to confidentiality agreements and the proprietary nature of the data involved, full numerical reproducibility of the associated manuscript is not possible within this public repository. This repository provides methodological transparency and partial public validation.

## Public Candidate (Subject to future release)
- **Literature Validation Set**: Derived from published sources. This is a candidate for future release subject to final review.

## Restricted (Not Publicly Available)
The following datasets and artifacts are proprietary and are strictly excluded from this repository:
- **Development Set**
- **Blind External Validation Set**
- **Additional Mexican PVT validation set**
- **Original Mexican PVT reports**
- **Trained models** derived from the proprietary Development Set.

The public code allows for methodological transparency, but does not include:
- fitted production model;
- proprietary training matrix;
- proprietary validation predictions.
